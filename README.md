# smart-agriculture-dbms
This project demonstrates the use of DBMS concepts such as tables, relationships, primary keys, foreign keys, normalization, and SQL operations. It is useful for reducing manual work, increasing productivity, and promoting smart farming practices through technology.
# Smart Agriculture Sensor DBMS
A mini DBMS project for managing farmers, fields, sensors, and sensor readings.

## Features
- Farmer management
- Field records
- Sensor management
- Sensor data readings
- SQL reports
