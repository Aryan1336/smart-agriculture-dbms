🌱 Smart Agriculture Sensor DBMS

A modern Database Management System project designed to support smart farming by storing, managing, and analyzing agricultural sensor data efficiently. This project helps farmers monitor field conditions such as soil moisture, temperature, humidity, and crop information for better decision-making.

---

📌 Project Overview

The Smart Agriculture Sensor DBMS is developed to automate agricultural data management using sensors installed in farm fields. It stores farmer details, field information, sensor records, and real-time readings in a structured relational database.

This system reduces manual work and helps improve crop productivity through data-driven farming.

---

🚀 Features

✅ Farmer record management
✅ Field and land details storage
✅ Sensor registration and monitoring
✅ Real-time sensor data collection
✅ Crop information management
✅ SQL-based reports and analysis
✅ Easy data retrieval and updates

---

🛠️ Technologies Used

- MySQL – Database Management
- SQL – Queries and Operations
- HTML/CSS – Frontend Interface
- GitHub – Version Control & Hosting

---

🗄️ Database Tables

- Farmer – Stores farmer details
- Field – Contains field location and area
- Sensor – Sensor type and status information
- Sensor_Data – Live sensor readings
- Crop – Crop details for each field
